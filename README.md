# Odoo Fleet Fuel Control Module

Custom Odoo module to register fuel refills and track fleet fuel costs inside **Odoo Fleet**.

## Why
Many fleets need structured fuel control to reduce waste and improve cost visibility.  
This module extends **fleet.vehicle** with a fuel log that supports cost tracking and future analytics.

## Features
- Fuel refill records linked to vehicles (Odoo Fleet)
- Cost calculation per refill
- Tree & Form views
- Basic validations
- Ready for reporting/BI (future extension)
